### JSnack 6
 Stampa il cubo dei primi 4 numeri, dove 4 è un numero indicato dall’utente.
 ## Logica
1. Assegnare una variabile con valore 4;
2. Un ciclo for che stampa numeri da 1 a 4
for (let i = 1; i <= 4; i++) {
    console.log(i ** 3)
}
